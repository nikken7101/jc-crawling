# jc-crawling

## Overview

[Japan Contest](http://japancontest.net) の結果ページをクロールして当選回数を調べます

## Requirement
* Python 3.x
* BeautifulSoup4
* lxml
* Pandas

## Usage
` $ python3 main.py`

## Output
<https://github.com/nikken7101/jc-crawling/blob/master/results.csv>
